# 前端动效与 3D 设计灵感库 (Design & 3D Resources)

这里是你提到的所有神仙网站合集。

## 🎯 现代 UI 与动效组件库

1. **[React Bits](https://www.reactbits.dev/)**
   - **干什么的**：收集高级 React 动效组件。
   - **特点**：物理引擎动效、光晕效果、文字动画。

2. **[Aceternity UI](https://ui.aceternity.com/)**
   - **干什么的**：高级 UI 组件库，专为 Next.js / React + Tailwind CSS 打造。
   - **特点**：发光边框、3D 卡片、视差滚动等高级交互组件。

3. **[Uiverse.io](https://uiverse.io/)**
   - **干什么的**：社区驱动的开源 UI 元素合集。
   - **特点**：纯 CSS/HTML 写的酷炫按钮、Checkbox、Loading 动画，不挑框架。

## 🎨 灵感与设计
4. **[Pinterest](https://www.pinterest.com/)**
   - **干什么的**：全球最大的设计师灵感库。

## 🎲 AI 3D 模型生成

5. **[Tripo 3D](https://www.tripo3d.ai/)**
   - **干什么的**：极速 AI 3D 模型工具。
   - **特点**：几秒钟生成 `.glb` 模型，适合原型或 Low-Poly 风格。

6. **[Meshy AI](https://www.meshy.ai/)**
   - **干什么的**：主打高质量贴图的 3D 生成 AI。
   - **特点**：模型细节丰富，材质光影好。
